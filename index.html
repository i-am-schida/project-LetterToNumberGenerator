Here's how it works:

Certain letters are replaced with an alternative character that resemebles it.

  • 'e' is replaced with '3'.
  • 'a' is replaced with '4'.
  • 'b' is replaced with '8'.
  • 'o' is replaced with '0'.
  • 'i' is replaced with '!'.
  • 'f' is replaced with 'ph' or 'PH'.
  • 's' is replaced with '$' or '5' randomly.
  • 'tfu' returns 'tfu', so 'stfu' returns '5tfu' or '$tfu'
  • "4 qu!ck 8r0wn ph0x jump$ 0v3r th3 l4zy d0g."
